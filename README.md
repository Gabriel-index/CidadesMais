# 🌆 Cidades+

**Tecnologia que resolve o que a cidade precisa.**

---

## 📌 Sobre o Projeto

> 📚 Este projeto foi desenvolvido como parte de um curso do SENAC Lapa Tito, pela turma **TII13**.
> 👨‍🎓 Alunos: **Gabriel Garcindo** e **Lucas Rodrigues**.

O **Cidades+** é um projeto de plataforma digital (site e futuramente aplicativo) que tem como objetivo facilitar a comunicação entre cidadãos e autoridades municipais para a resolução de problemas urbanos.

A proposta vai além de apenas registrar problemas — o projeto incentiva a participação ativa da comunidade na criação de soluções criativas, sustentáveis e colaborativas para melhorar a cidade.

---

## 🎯 Objetivo

Permitir que qualquer cidadão possa:

* Registrar problemas urbanos (ex: buracos, iluminação, acessibilidade)
* Sugerir soluções
* Interagir com outros usuários
* Acompanhar o andamento das solicitações

Além disso, o projeto busca promover aprendizado prático sobre tecnologia, urbanismo e cidadania.

---

## 🚀 Funcionalidades

✔️ Registro de problemas urbanos com descrição
✔️ Possibilidade de envio de imagens (futuramente)
✔️ Sistema de apoio (curtidas/votos)
✔️ Comentários e interação entre usuários
✔️ Acompanhamento de status (em análise, em andamento, resolvido)
✔️ Dashboard público (planejado)
